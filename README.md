# Sauce Demo Playwright Automation Framework

## Overview

This project is a UI Test Automation Framework built using:

- Playwright
- JavaScript
- Page Object Model (POM)
- Playwright Fixtures
- Hooks (BeforeEach & AfterEach)
- HTML Reporting

---

# Installation

Clone the repository:

```bash
git clone git@github.com:maxxy4real/sauce-demo-playwright.git
```

Navigate to project directory:

```bash
cd sauce-demo-playwright
```

Install dependencies:

```bash
npm install
```

Install Playwright browsers:

```bash
npx playwright install
```

# Test Tags

The framework supports test tagging.

## To run all Sanity Tests

```bash
npm run test:sanity
```

## To run all  Regression Tests

```bash
npm run test:regression
```

## Run All Tests

```bash
npm run test
```

---

## Run All Tests in Browser

```bash
npm run test:headed
```
